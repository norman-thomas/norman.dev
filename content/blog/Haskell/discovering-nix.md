# Installation

## Prerequisites

- macOS (11.1 Big Sur)
- Terminal

## Download and Install

```sh
sh <(curl -L https://nixos.org/nix/install) --no-daemon
```

This will install Nix into the `/nix` folder.

Check it is properly installed by running

```sh
nix --version
```
